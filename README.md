# 🎓 Hari Academic Portfolio

A modern, responsive academic portfolio dashboard designed to showcase academic performance, technical skills, internship experience, and projects.

## 📌 About The Project

This portfolio was developed to present my academic achievements, semester results, technical skills, internship experience, and projects in a clean and professional layout suitable for placement and professional showcase.

It includes interactive UI components and a structured academic dashboard.

---

## 🚀 Features

- 📊 Semester-wise Academic Dashboard
- 📈 Animated Skill Analytics Chart
- 🎓 Academic Record Table (SGPA Included)
- 💼 Internship Experience Section
- 🚀 Projects Showcase
- 📄 Certificate Viewer
- 🌐 Responsive Design
- ✨ Modern UI with Flip Cards

---
